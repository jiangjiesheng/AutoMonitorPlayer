# NiceVieoPlayer
## 用IjkPlayer/MediaPlayer + TextureView/SurfaceView封装，完美切换全屏、小窗口播放可拖动,支持第三方播放器。
### Usage
下载niceviewoplayer库，在AndroidSutio中作为Module添加依赖。
### 遗留问题
使用全局悬浮窗, 在切到主界面时切换窗口类型会导致窗口消失。因为这时应用不可见，无法显示全屏窗口
